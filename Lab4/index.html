¿Qué diferencias y semejanzas hay entre Java y JavaScript?

Diferencias:
Tipo de lenguaje: Java es un lenguaje de programación orientado a objetos, mientras que JavaScript es un lenguaje interpretado y orientado a objetos y prototipos.
Aplicación: Java se utiliza principalmente en desarrollo de aplicaciones de escritorio, web empresarial y móvil, mientras que JavaScript se utiliza principalmente en desarrollo web.
Compilación vs. Interpretación: Java se compila en bytecode ejecutable en la JVM, mientras que JavaScript es interpretado por el navegador.
Semejanzas:
Ambos son lenguajes de programación de propósito general.
Ambos tienen sintaxis similares a C.
Ambos son ampliamente utilizados en la industria del software.

¿Qué métodos tiene el objeto Date? (Menciona al menos 5*)
getDate(): Devuelve el día del mes (1-31).
getMonth(): Devuelve el mes (0-11).
getFullYear(): Devuelve el año.
getHours(): Devuelve la hora (0-23).
getMinutes(): Devuelve los minutos (0-59).
¿Qué métodos tienen los arreglos? (Menciona al menos 5*)
push(): Agrega uno o más elementos al final del arreglo.
pop(): Elimina el último elemento de un arreglo.
shift(): Elimina el primer elemento de un arreglo.
unshift(): Agrega uno o más elementos al principio de un arreglo.
splice(): Cambia el contenido de un arreglo eliminando o reemplazando elementos existentes y/o agregando nuevos elementos.
